Import from Plex

Connect to your Plex server to import your watch history.

Important: local IPs won't work

NextUp runs in the cloud, so it can't reach a Plex server on your home network (addresses like 192.168.x.x or 10.x.x.x). You need either:

  • A Plex Remote Access URL — a plex.direct hostname. Enable it in Plex Settings → Remote Access.
  • Or a Plex server hosted publicly (e.g. on a VPS).

How to find your server URL and token

  1. Open Plex Web App and sign in
  2. Open any media item and click Get Info (or "...")
  3. Click View XML at the bottom
  4. Copy the full URL from the address bar up to :32400 — that's your server URL. Look for a plex.direct hostname, not a 192.168 address.
  5. In the same URL, find X-Plex-Token= and copy the value after it — that's your token.
